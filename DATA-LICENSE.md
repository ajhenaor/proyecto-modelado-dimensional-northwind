# Datos Northwind

Los archivos CSV en `datasets/northwind/` y los seeds derivados en `seeds/` provienen del conjunto de datos de ejemplo **Northwind**, publicado por Microsoft como material de demostración.

- Referencia: [Microsoft SQL Server samples](https://github.com/Microsoft/sql-server-samples)
- Uso: redistribución con fines educativos, alineada con el uso habitual del sample Northwind en documentación y tutoriales.

El código dbt de este repositorio (modelos, macros, tests) está bajo la licencia MIT — ver `LICENSE`.
